## 如何理解这三个名字

- `Tools Tutorial` 是上位概念，类似“目录”。
- `Claude Tutorial` 与 `Psmux Tutorial` 是下位概念，类似“目录下的具体文章”。

如果用集合关系来表达，可以写成：

$$
\text{Tools Tutorial} \supset \{\text{Claude Tutorial},\ \text{Psmux Tutorial}\}
$$

这里的 `\supset` 表示“包含”。它强调的是导航组织关系，而不是把两篇教程机械拼接成一篇长文。
